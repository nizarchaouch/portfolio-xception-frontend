![Vue.js](https://img.shields.io/badge/Vue.js-Frontend-42b883)
![Vuetify](https://img.shields.io/badge/Vuetify-UI-1867C0)
![Vuex](https://img.shields.io/badge/Vuex-State%20Management-764ABC)
![Status](https://img.shields.io/badge/Status-Academic%20Project-blue)

This repository contains the frontend of **Portfolio Xception**, a full-stack web platform designed to help candidates create professional portfolios, explore job opportunities, and apply for jobs through one unified system.

The platform also provides recruiters with tools to publish job offers, manage applications, and search for candidates, while administrators can supervise users, offers, and portfolio templates.


## Frontend Features

- User authentication interfaces
- Candidate profile management
- Recruiter dashboard and management pages
- Admin dashboard and management pages
- Job offers listing and filtering
- Portfolio template browsing
- Portfolio editor interface
- CV upload interface
- Application tracking pages
- Responsive user interface

## Tech Stack

- Vue.js
- Vuetify
- Vuex
- HTML5
- CSS
- JavaScript

## Getting Started

### Prerequisites

Make sure you have installed:

- Node.js
- npm

### Clone the repository

```bash
git clone https://github.com/nizarchaouch/portfolio-xception-frontend.git
cd portfolio-xception-frontend
```

### Install dependencies

```bash
npm install
```

### Environment Variables

Create a `.env` file in the root directory and add the required frontend variables:

### Run the project

```bash
npm run serve
```

### Build for production

```bash
npm run build
```
